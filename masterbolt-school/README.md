# Masterbolt - Product Page

A beautiful, modern product page for Masterbolt showcasing all features and functionalities. Built with Nuxt 3, Three.js, and Tailwind CSS.

## Features

- **3D Interactive Hero Section** - Engaging 3D particle system and geometric shapes
- **Feature Showcases** - All key functionalities from the pitch document
- **Responsive Design** - Works beautifully on all devices
- **Modern UI** - Glass morphism effects, gradients, and smooth animations

## Key Sections

1. **Hero Section** - 3D background with interactive particles
2. **Core Philosophy** - Socratic Method and Feynman Technique
3. **Generative Visuals** - Custom images and videos on-demand
4. **Adaptive Learning** - Inclusivity features for all learners
5. **Assessment & Gamification** - Smart quizzes and motivation system
6. **Evolving Models** - Beyond K-12 features (Code onboarding, API tutorials, etc.)
7. **Sutradhar Technology** - The AI orchestrator that powers everything
8. **Content Quality** - OpenStax curriculum highlights

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The site will be available at `http://localhost:5000`

## Tech Stack

- **Nuxt 3** - Vue.js framework
- **Three.js** - 3D graphics library
- **GSAP** - Animation library
- **Tailwind CSS** - Utility-first CSS framework
