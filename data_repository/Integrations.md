# Integrations

## Supported Platforms

Slack: Post video links to channels, receive notifications.

Google Calendar: Schedule video recordings, sync events.

GitHub: Create issues from video annotations, link PRs.

## Setup Instructions

Slack integration requires OAuth token from workspace admin.

Calendar integration uses Google OAuth, requires calendar permissions.

GitHub integration needs personal access token with repo scope.

## API Access

Business plans include REST API access.

API rate limits: 1000 requests per hour for Business, 100 for Pro.

Webhooks available for video processing completion events.
